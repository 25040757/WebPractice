#day 1 Personal Bio Card

A simple HTML bio card page built as part of my web development learning journey.

## What it includes
- Personal photo loaded from a local file
- Name and personal introduction using headings and paragraphs
- Fun facts list using an unordered list

## What I learned
- Basic HTML page structure using DOCTYPE, head and body
- How to use headings h1 and h2 correctly
- How to embed a local image using the img tag and src attribute
- The importance of the alt attribute on images
- How to use ul and li to create a list
- Why headings must never be placed inside list tags

## How to run
Download the file and the image, keep them in the same folder,
then open index.html in any browser. No setup needed.

#day 2 Recipe Page — Beef Stew

A simple HTML recipe page built as part of my web development learning journey.

## What it includes
- Semantic HTML structure using header, main and footer
- Meal image loaded from a URL
- Ingredients list using an unordered list

## What I learned
- Difference between ul and ol and when to use each
- How to properly nest li items inside a list
- How to use semantic tags like header, main and footer
- How to embed images using a URL
